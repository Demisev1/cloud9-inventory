## 🚀 Overview

This app helps streamline inventory restocking for Cloud 9 Vapes by analyzing CSV files of sales and stock.  
It calculates what needs to be reordered using advanced logic, shows stock warnings, and provides quick export functionality.

---

## 🧠 Features

- 📦 **Advanced Restock Logic**: Calculates reorder qty using:

- ⚠️ Negative Stock Detection
- 📊 Category-Based Sorting + Live Search
- 🎛 Adjustable Reorder Multiplier and Buffer %
- 🌒 Toggle Light / Dark Theme
- 📥 CSV Upload + Reorder Download
- 📱 Fully responsive (mobile + desktop)
- 🧾 Versioned and maintained by Shane Warren

---

## 📁 How To Use

1. Open `index.html` in your web browser.
2. Upload your inventory `.csv` file (must contain: `Product name`, `Category`, `Qty`, `Last Sales/7 Days`)
3. Adjust the **Multiplier** and **Buffer %** if needed
4. Click `Analyze`
5. View restock items and download reorder list (CSV)
